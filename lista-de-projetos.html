SpotDy

Full Stack

A proposta desse projeto foi para que fosse criada uma API de músicas e playlists que conversasse com o banco de dados, e uma aplicação em React que consumisse essa API. Podendo criar músicas, playlists, adicionar músicas a essas playlists e mostrar detalhes delas. Acrescentei um player customizado, que neste momento está sendo desenvolvido.
